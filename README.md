# E-commerce Sales Analysis

## Project Overview
This project analyzes an e-commerce dataset using spreadsheet tools (ODS format) to identify sales trends, category performance, and actionable business insights.

## Objectives
- Analyze monthly sales trends and seasonality
- Evaluate product category performance
- Identify underperforming segments
- Detect and analyze outliers
- Generate business recommendations

## Tools Used
- ODS (OpenDocument Spreadsheet)
- Data Analysis in spreadsheet environment
- Data Visualization (charts and dashboard)

## Key Insights
- Sales peaks observed in July and December
- Clothing category showed the lowest profitability
- Country-level performance differences identified
- Outliers detected and evaluated

## File Included
- `ecommerce_sales_analysis.ods` – full dataset analysis, calculations, and dashboard

## Business Value
This project demonstrates the ability to:
- Extract insights from structured data
- Identify trends and anomalies
- Translate data findings into business recommendations
